# Todo-app-using-MERN
A minor project to get a better understanding on MERN tech stack by creating Todo web application
In this application I used MVC framework

### Libraby Used
- ejs
- express
- mongoose

#### Install libraries By using below commands

```
// for ejs
npm install ejs

// for express
npm install express

// for mongoose
npm install mongoose
```
